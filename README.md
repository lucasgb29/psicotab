# psicotab
Psicotab is a fullstack web platform designed to promote scientifically validated mental health content and connect users with qualified psychology professionals.

The platform ensures that only verified specialists can publish content, addressing the widespread issue of misinformation in mental health discussions on social media. Additionally, it facilitates interaction between professionals and the general public while helping patients find psychologists that best match their needs.
